# Tank Fighter

A 4-player LAN Tank Game for Android built with Godot 4.x

## Features
- Host-client LAN multiplayer using Godot's MultiplayerAPI
- Turn-based artillery gameplay
- Destructible terrain
- JSON-based weapon system
- One-thumb touch controls
- Android haptic feedback
- Automated builds via GitHub Actions
